## 📝 技术实现

### 事件监听

```javascript
// 为每个选项卡按钮添加点击事件
tabButtons.forEach(button => {
    button.addEventListener('click', () => {
        const targetTab = button.getAttribute('data-tab');
        
        // 1. 移除所有活动状态
        tabButtons.forEach(btn => btn.classList.remove('active'));
        tabContents.forEach(content => content.style.display = 'none');
        
        // 2. 激活当前选项卡
        button.classList.add('active');
        
        // 3. 显示对应内容
        document.getElementById(`${targetTab}-tab`).style.display = 'block';
    });
});
```

### HTML 结构

```html
<!-- 按钮 -->
<button class="tab-button" data-tab="folders">主页</button>
<button class="tab-button" data-tab="project-management">项目</button>
<button class="tab-button" data-tab="recent">最近</button>

<!-- 内容 -->
<div id="folders-tab" class="tab-content">主页内容</div>
<div id="project-management-tab" class="tab-content">项目内容</div>
<div id="recent-tab" class="tab-content">最近内容</div>
```
