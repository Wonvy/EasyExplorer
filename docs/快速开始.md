## 💡 使用提示

### 选择前端实现

您有两种选择：

#### 选项 1: 使用适配器 (最小改动)

在 `index.html` 中：
```html
<script src="tauri-adapter.js"></script>
<script src="renderer.js"></script>
```

**优点**: 原代码几乎不用修改
**缺点**: 可能有兼容性问题

#### 选项 2: 使用 Tauri 原生实现 (推荐)

在 `index.html` 中：
```html
<script src="renderer-tauri.js"></script>
```

**优点**: 性能最佳，完全兼容
**缺点**: 需要完善功能实现
