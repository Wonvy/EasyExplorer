# 快速开始 - 转化版

## 🚀 立即开始使用

### 1. 启动应用
```bash
cd D:\Code\Electron\EasyExplorer
npm run dev
```

### 2. 核心功能
应用已包含以下完整功能：

#### 📁 文件浏览
- 浏览文件和文件夹
- 多种视图模式
- 文件排序

#### 📅 日历视图
1. 点击侧边栏"项目"选项卡
2. 点击"日历"按钮
3. 查看当月项目
4. 使用滚轮快速切换月份
5. 双击文件夹进入

#### 📊 年报视图
1. 点击侧边栏"项目"选项卡
2. 点击"年报"按钮
3. 查看全年12个月的项目
4. 拖拽或滚轮浏览时间轴
5. 单击查看预览，双击打开

#### 📋 文件操作
- **复制**: 选中文件，按 `Ctrl+C`
- **剪切**: 选中文件，按 `Ctrl+X`
- **粘贴**: 按 `Ctrl+V`
- **刷新**: 按 `F5`

#### ⭐ 收藏夹
1. 右键文件夹
2. 选择"添加到收藏"
3. 在侧边栏"主页"查看收藏

#### 📂 文件夹分组
1. 点击侧边栏"项目"选项卡
2. 点击"+"创建新分组
3. 右键文件夹添加到分组
4. 右键分组进行管理

#### 🎨 主题切换
1. 点击左下角设置图标
2. 点击"切换主题"
3. 在亮/暗主题间切换

## ⚠️ 注意事项

### 配置要求
使用日历和年报功能前，需要先配置项目路径：
1. 点击设置图标 → "设置"
2. 在设置页面配置年份对应的项目路径
3. 例如：`2025` → `D:\Projects\2025`

### 目录结构
建议的项目目录结构：
```
D:\Projects\2025\
├── 1月\
│   ├── 0101-元旦项目
│   ├── 0115-中旬项目
│   └── ...
├── 2月\
│   └── ...
...
└── 12月\
    └── ...
```

## 🆘 常见问题

### Q: 日历显示"未设置项目文件夹"？
**A**: 请在设置中配置对应年份的项目路径。

### Q: 快捷键不工作？
**A**: 确保焦点在应用窗口内，不在输入框等元素上。

### Q: 分组保存失败？
**A**: 检查浏览器控制台错误，可能是 localStorage 配额问题。

### Q: 年报加载慢？
**A**: 正常现象，需要读取12个月的文件夹数据，请耐心等待。
