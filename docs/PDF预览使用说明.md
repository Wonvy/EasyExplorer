# PDF 预览功能 - 快速使用指南

## 🎉 问题已修复！

PDF 文件预览功能已经实现，现在可以直接在预览面板中查看 PDF 内容了！

## 🚀 立即开始使用

### 1. 启动应用
```bash
npm run dev
```

### 2. 预览 PDF 文件
1. 在文件浏览器中找到任意 PDF 文件
2. 点击该 PDF 文件
3. 查看右侧预览面板

## ✨ 新功能特性

### 📄 实时预览
- 无需打开外部程序
- 直接在预览面板中显示 PDF 内容
- 高质量渲染，清晰可读

### 📖 多页浏览
- 对于多页 PDF，底部会显示翻页控制
- 使用 ◀ 和 ▶ 按钮切换页面
- 实时显示当前页码（例如：第 1 页 / 共 5 页）

### ⚡ 快速加载
- 智能加载机制
- 显示加载进度提示
- 不会卡死界面

### 🛡️ 错误处理
- 如果 PDF 无法预览，会显示友好的错误提示
- 始终提供"使用默认程序打开"的备用选项

## 🎮 操作说明

### 查看 PDF
1. 点击任意 PDF 文件
2. 预览面板自动显示内容

### 翻页（多页 PDF）
1. 点击 ▶ 按钮查看下一页
2. 点击 ◀ 按钮查看上一页
3. 中间显示当前页码

### 外部打开
1. 点击"使用默认程序打开"按钮
2. 系统默认 PDF 阅读器将打开该文件

## 💡 使用技巧

### 适合预览的 PDF 类型
- ✅ 普通文本 PDF
- ✅ 包含图片的 PDF
- ✅ 多页文档
- ✅ 小于 10MB 的文件

### 不适合预览的情况
- ❌ 加密或受保护的 PDF
- ❌ 超大文件（> 50MB）
- ❌ 损坏的 PDF 文件

**对于不适合预览的文件，请使用"外部打开"功能。**

## 🔧 故障排除

### 问题：预览显示空白
**解决方案**：
1. 检查 PDF 文件是否损坏
2. 尝试用其他 PDF 阅读器打开
3. 查看浏览器控制台（F12）是否有错误信息

### 问题：加载很慢
**解决方案**：
1. 大文件（> 10MB）加载需要时间，请耐心等待
2. 如果超过 10 秒还未加载，建议使用外部打开

### 问题：翻页按钮不工作
**解决方案**：
1. 刷新应用重试
2. 检查是否是单页 PDF（单页 PDF 不显示翻页按钮）

### 问题：显示错误信息
**解决方案**：
1. 阅读错误提示信息
2. 使用"外部打开"按钮作为备用方案
3. 如果是重要问题，请报告给开发团队

## 📊 性能参考

|---------|---------|-------------|
| 单页文档 | < 1MB   | < 1秒       |

| 中等文件 | 5-10MB  | 3-5秒       |
| 大文件  | > 10MB  | > 5秒       |

*实际性能因系统配置而异*

1. **快速预览**：对于小文件，直接使用预览功能
2. **详细阅读**：对于需要长时间阅读的 PDF，建议使用外部打开

4. **多任务**：预览 PDF 的同时可以继续浏览其他文件
